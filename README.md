Simple cryptocurrency portfolio tracker with Windows95 UI

Featuring:
* mobile-first design
* infinite scroll
* coin search
* theming, background and font settings
* client side only data storage
* crypto data from [CryptoCompare API](https://min-api.cryptocompare.com/)
